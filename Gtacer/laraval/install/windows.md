1. 安装基本的php开发环境

2. 安装 Composer

3. 使用 Composer 安装 Laravel
    - 进入到需要下载 Laravel 的目录  
	`cd /path/WWW`
	- 执行如下 Composer 命令  
	`composer create-project laravel --prefer-dist`
    - Laravel 环境部署完成，如果遇到报错，请尝试以下指令  
    `composer self-update`

4. 通过浏览器访问 Laravel 的引导页
    - 引导页面所在地址为  
    `\public\index.php`
    
5. 将网站根目录指向 `\public` 目录
